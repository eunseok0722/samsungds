Product name placement: Memory NAND & B2C products

업계의 특성 및 표준 용어에 따라 “제품 카테고리”는 제품명 / 모델 전후로 사용 가능하나
메모리 사업부, NAND 및 B2C 제품은 제품 카테고리 > 제품명 순으로 사용을 권장합니다.
하기를 참고 하십시오.

  1. Samsung Sharp Sans Bold 서체 사용
  2. 제품 네이밍 구조 활용 시 화이트 또는 블랙 색상 사용
  3. 제품 카테고리 및 제품명 첫 문자는 대문자 사용
  4. 최소 사이즈 이하 축소 불가(인쇄물 3mm 높이, 디지털 8 픽셀)
  5. 전체 자간 트래킹 +10 pt 디자인프로그램
    사용시 설정 필요
    1. * Mircrosoft Word 및 Powerpoint
    2. "문자 간격" 에서 변경 가능

Product name placement: Dos / Don'ts

Do

  • Memory NAND : 제품명/모델명을
    볼드로 강조할 수 있습니다.
  • Memory DRAM : 제품명/모델명을
    볼드로 강조할 수 있습니다.
  • S.LSI : 제품명 및/또는 식별자만
    사용할 수 있습니다.
  • 파운드리

Do not

  • 제품 네이밍을 사용할 때, 이탤릭체
    또는 굵기가 다른 글꼴을 조합
  • 제품명 중간에 빈칸을 삽입
  • 제품명의 일부로서 제품 서술자를 포함
  • 굵기가 다른 Samsung Sharp Sans를 사용

Do not

  • 대문자와 소문자를 임의로 사용
    [1-2.1 제품 네이밍 아키텍처]를 참고하십시오.
  • 색상 또는 그라데이션을 적용
  • 특정 단어를 확대 또는 축소
  • TM 심볼에 윗첨자가 아닌 글자를 사용
